# E-Commerce Sales Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard developed to analyze e-commerce sales performance. It provides insights into revenue, profit, customer segments, product categories, shipping methods, and regional sales performance through interactive visualizations.

---

## Business Objectives

- Monitor overall sales performance
- Track profit and profit margin
- Identify top-performing and low-performing products
- Analyze customer purchasing behavior
- Evaluate regional sales performance
- Compare shipping mode efficiency
- Support data-driven business decisions

---

## Dashboard Features

- Sales KPI Dashboard
- Profit Analysis
- Quantity Analysis
- Profit Margin Analysis
- Product Category Analysis
- Customer Segment Analysis
- Regional Sales Analysis
- State-wise Sales Map
- Shipping Mode Analysis
- Top 5 Products
- Bottom 5 Products
- Interactive Filters and Slicers
